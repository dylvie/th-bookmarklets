## ğŸš€ How to Add a Bookmarklet to Your Browser

Follow these steps to install any bookmarklet from this repository:

### Chrome / Edge / Firefox

1. Open your browser.
2. Show the bookmarks bar (if hidden):
   - **Chrome/Edge**: `Ctrl+Shift+B`
   - **Firefox**: `View > Toolbars > Bookmarks Toolbar`
3. Create a new bookmark:
   - Right-click the bookmarks bar and choose **Add page** or **Add bookmark**.
4. In the **Name** field, enter a label (e.g., `Scan with VirusTotal`).
5. In the **URL** field, paste the entire JavaScript code from the bookmarklet file.
6. Save the bookmark.

### Safari (macOS)

1. Enable the **Favorites Bar** via `View > Show Favorites Bar`.
2. Open any webpage.
3. Create a new bookmark from the page (`Cmd+D`).
4. Edit the bookmark:
   - Rename it.
   - Replace the URL with the JavaScript code.
5. Save and use it from the Favorites Bar.
